# React Portfolio Website

## Description
This is my personal portfolio website built using React.js to showcase my skills, projects, resume, and contact information.

## Technologies Used
- React.js
- HTML5
- CSS3
- JavaScript

## Features
- Responsive Portfolio Website
- About Me
- Skills
- Projects
- Resume
- Contact Form
- Lazy Loading using React.lazy()
- Production Build Optimization

## Projects
### Rentomatic
- Real Estate Website
- React + ASP.NET + SQL Server

### Weather App
- React + OpenWeather API

## Installation

```bash
npm install
npm start
```

## Production Build

```bash
npm run build
```
